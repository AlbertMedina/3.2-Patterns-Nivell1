# Patterns

## Description - Exercise statement
Three levels of exercises to work with design patterns.

### Level 1
Using Builder pattern to implement a system to build pizzas in a pizza store.

### Level 2
Using Observer pattern to create a system where a stockbroker notifies multiple brokerages about stocks state.

### Level 3
Using Callback pattern to implement a payment gateway that holds different payment methods.

## Technologies used
- Oracle OpenJDK 21.0.8
- Gradle 8.13
- IntelliJ IDEA Comunity Edition

## Requirements
- Oracle OpenJDK 21.0.8
- Gradle 8.13
- Java IDE

## Installation
1. Clone repository (https://github.com/AlbertMedina/3.2-Patterns-Nivell3.git).
`git clone https://github.com/AlbertMedina/3.2-Patterns-Nivell3.git`
2. Import project to Java IDE.

## Execution
1. Open project with Java IDE.
2. Locate the three packages (level1, level2, level3).
3. Inside each package, locate and open the file "Main.java".
4. Run each one from the IDE.
5. Tests for each level can also be found in the test folder.
